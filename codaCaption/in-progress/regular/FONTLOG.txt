Basic Font Information

Coda is a display webfont designed to be used freely across the internet by web browsers on desktop computers, laptops and mobile devices.

This release supports the following 
Unicode ranges: Latin, Latin-Extended

There are 4 Source files:

1. src/Coda.sfd Original Source Files in postscript curves.
2. src/Coda.ufo Source Files in UFO format.
3. src/Coda.otf OpenType font format.

To contribute to the project contact Vernon Adams
at vern@newtypography.co.uk and use the gitHub repository at https://github.com/vernnobile/CodaFont
